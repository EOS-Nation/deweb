> **deWeb** common library used for Typescript

## Installation

Using Yarn:

```bash
yarn add deweb
```

or using NPM:

```bash
npm install --save deweb
```

## Quick Start

```ts
import { stringToHash } from "deweb"

const id = stringToHash("deniscarrier@deweb.eosnation.io");
//=> 8945370747656800747
```
